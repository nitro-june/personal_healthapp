# Personal Mental-Health Tracker

![Logo](logo.jpg)

An easy way to keep track of mental health, without giving your data away to people you don't know.

---

The mental-health tracker uses python and sqlite to localy save the data you input. Then this data is put on the convenient dashboard, which you can always customize the layout of yourself. Many of the trackables are similar to commonly used screening tests for depression or anxiety.
